## IPTV Strean Utility

Cabernet allows control of IPTV streams. Plugins supports DaddyLive, Pluto TV, XUMO, M3U/XMLTV.XML files (SamsungTV, STIRR, DistroTV, Plex TV)

### Full Documentation / Install Instructions

https://docs.tvheadend.org/
